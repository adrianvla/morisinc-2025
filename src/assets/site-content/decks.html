Overview

This is a LaTeX Flashcard app, with Spaced Repetition and with LaTeX support.

It's open source too, which means that the source code can be found [btn link="https://github.com/adrianvla/decks"]here[/btn].

You can use it online on [btn link="https://decks.morisinc.net/"]decks.morisinc.net[/btn]. If you have trouble, please visit the [a href="https://decks.morisinc.net/help"]Help page[/a].

Features

  • LaTeX Support
  • Normal flashcard mode (like Quizlet)
  • Self-hosted database (requiring each user to enter their own firebase api-key)
  • Cross-platform (online WebApp)
  • Flashcard Deck Import & Export
  • Dark & Light mode

Why did I make this?

I was using [a href="https://www.studysmarter.de/"]StudySmarter[/a] to prepare for exams. It was buggy when I was creating flashcards at a very fast pace, not waiting before creating another one.

2 days before my Calculus I Part 1 exam, StudySmarter decided it was a good idea to make half of my flashcards disappear for no reason at all. Well, the reason was that quickly creating flashcards on that platform was already unstable.

This made me roll out my first prototype of this app in 1 week, which I used for the second part of my Calculus I exam (it was split in 2 parts).

I never used StudySmarter again, and advise everyone to stay away from it because of it's bugs.

Unlike that app, Decks supports fast flashcard creation and is made to protect your flashcards at all costs.
