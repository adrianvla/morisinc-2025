A geiger counter on M5 Stack

A smart geiger counter built on M5 Stack with a radiation map that uses OpenStreetMap, with wifi to log the radiation on the map. To build your own, download the repository from [btn link="https://github.com/adrianvla/M5-Stack-Geiger-Counter"]here[/btn].

M5 Stack

Installation

  • Download the repo
  • Replace M5.BtnA.isPressed() at line 112 of src/main.cpp to match your 'tube tick' for radiation counting.
if (M5.BtnA.isPressed()) {
						if (flag_tick) {
						  signalHandler();

						  flag_tick = false;
						}
					  } else {
						flag_tick = true;
					  }
  • The 2nd button (button B) resets the counter
  • Compile the project for M5Stack core, I've used PlatformIO on Visual Studio Code instead of Arduino.
  • Program the M5 stack with the compiled code
  • Turn on the M5 Stack

How To use

  • Turn on M5Stack by pressing the red button.
  • Connect your device to M5Stack_setup wifi network and follow some instructions on the m5stack screen
  • Go to http://192.168.4.1 on your device
  • You're done!
